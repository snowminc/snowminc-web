# snowminc-web
A website to practice web development - http://www.snowminc.com

The website has two purposes:

 1. Be my portfolio and a way to express myself as a developer.
 2. Be the testbed for every new web development concept that I learn to be more versatile in programming.
 
The general idea I had for this project is simply to add, add, add. I may write an entire page using one framework, then write the same page again with another framework. I may write random webapps for random purposes. I may write up a blog to detail projects that I've worked on, or even other webapps written on this website.

Overall, my goal is just to continue learning independently as a developer to keep myself fresh, and having an actual means to that end is more motivating than just waiting to think of a completely isolated idea for a project.

The website is hosted on [Heroku](https://www.heroku.com/).
